## File Naming Format
Use the user ID from `data.js` as the image filename:

- **Format**: `{USER_ID}.jpg` or `{USER_ID}.png`
- **Example**: `U001.jpg`, `U002.png`, etc.

## Recommended Image Specifications
- **Format**: JPG or PNG
- **Dimensions**: 150x150px (square, 1:1 aspect ratio)
- **File Size**: Keep under 200KB for fast loading
- **Style**: Professional headshots or profile photos
- **Optimization**: Compress images before uploading

## Usage in Application
Profile images are used in:
- Profile page header
- Review cards (showing reviewer avatar)
- Owner dashboard
- Comments and responses
